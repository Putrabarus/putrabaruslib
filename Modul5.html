Cara Menampilkan Data dalam JavaScript

Ada beberapa cara untuk menampilkan data dalam JavaScript di halaman web:

  1. Menggunakan innerHTML: Kita dapat mengubah konten HTML dari suatu elemen dengan properti innerHTML.
  2. Menggunakan document.write(): Metode ini menulis langsung ke dokumen HTML, tetapi sebaiknya dihindari karena dapat menimpa konten yang ada.
  3. Menggunakan console.log(): Berguna untuk debugging, menampilkan data di konsol browser.
  4. Menggunakan alert(): Menampilkan data dalam bentuk pop-up alert.
