    HTML (HyperText Markup Language) adalah bahasa pemrograman dasar yang digunakan untuk membangun struktur sebuah halaman web. Bayangkan HTML sebagai kerangka dari sebuah rumah. HTML mendefinisikan elemen-elemen seperti judul, paragraf, gambar, tautan, dan bagian-bagian lainnya. Setiap elemen dalam HTML diapit oleh tag yang diawali dengan tanda kurung siku (<) dan diakhiri dengan tanda kurung siku (/>). Dengan HTML, kita dapat mengatur tata letak dasar dari sebuah halaman web, namun untuk membuat tampilan yang lebih menarik dan profesional, kita membutuhkan CSS.

    CSS (Cascading Style Sheets) adalah bahasa desain yang digunakan untuk mengatur tampilan dan gaya dari sebuah halaman web. Jika HTML adalah kerangka rumah, maka CSS adalah cat, perabotan, dan dekorasi yang membuat rumah tersebut terlihat indah dan menarik. CSS memungkinkan kita untuk mengatur warna, font, ukuran, tata letak, dan efek visual lainnya pada elemen-elemen HTML. Dengan CSS, kita dapat membuat halaman web menjadi lebih responsif, interaktif, dan sesuai dengan keinginan kita.
